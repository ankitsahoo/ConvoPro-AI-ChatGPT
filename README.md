# ConvoPro-AI-ChatGPT

# Tech Stack Used

- **Streamlit** - User Interface
- **Ollama** - LLM Provider
- **MongoDB** - Store Past Conversations
- **Docker** - For Deployment
- **AWS EC2** -  Application Host

# ConvoPro Features
- User Friendly Interface
- Configurable LLM
- Store Past Conversations
- Local/ Private Server - Secure
